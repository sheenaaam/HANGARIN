# Hangarin: Task & To-Do Manager

Hangarin is a simple web application built with **Django** that helps users:

- Organize daily tasks
- Manage priorities
- Add notes
- Break down large goals into smaller subtasks

It features a responsive **Bootstrap 5 dashboard** for tracking task status and quick management.



## Features

- Add, edit, delete, and list:
  - Tasks
  - SubTasks
  - Categories
  - Priorities
  - Notes
- Dashboard showing:
  - Total Tasks, Pending Tasks, Completed Tasks
  - Total SubTasks
  - Total Notes
- Responsive design using Bootstrap 5
- Admin interface for managing all objects
- Search and filter for Tasks and SubTasks
